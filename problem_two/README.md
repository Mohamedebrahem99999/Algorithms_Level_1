# Problem #2: Read and Print Name

## Problem Description

Write a program to ask the user to enter his/her name and print it on the screen.

---

## 🛠️ Algorithm Steps (Logic)

To solve this problem, we need to perform two main steps:

1. **Input/Read:** Prompt the user to enter their name and store it in a variable (e.g., `Name`).
2. **Output:** Print the value stored in the `Name` variable to the screen.

---

## 📊 Flowchart Logic

1. **Start**
2. **Input:** `Read Name`
3. **Output:** `Print Name`
4. **End**
