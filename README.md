# Algorithms and Problem Solving - Level 1

---

## Description
This repository contains my solutions for the "Algorithms & Problem Solving Level 1" course from Programming Advices (Abu-Hadhoud). The project aims to build a solid foundation in logic building and algorithmic thinking using C++.

---

## Course Content
The repository is organized by problems, where each folder or file represents a specific challenge from the course. These problems cover:
* Basic Input/Output operations.
* Conditional Statements (If-Else, Switch).
* Loops and Iterations.
* Functional Programming and Logic Reuse.

---

## Problem Examples
One of the key problems included is **Problem #5: Hire a Driver (Case 2)**, which demonstrates complex conditional logic:
* **Priority Rule:** A candidate with a recommendation is hired immediately.
* **Standard Rule:** Otherwise, the candidate must be older than 21 and possess a valid driver's license.

---

## How to Use
1. Clone the repository to your local machine.
2. Navigate to the specific problem folder.
3. Compile the C++ file using any standard compiler (like GCC or MSVC).
4. Run the executable to test the logic.

---

## Technologies Used
* Language: C++
* Logic Design: Flowcharts and Pseudocode.
