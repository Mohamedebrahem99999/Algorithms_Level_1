# Problem #9: Sum of 3 Numbers

## 📝 Problem Description
The goal of this program is to prompt the user to enter three separate numbers, calculate their total sum, and then display the result on the screen.

**Example:**
- If the user enters: `10`, `20`, and `30` 
- The Output will be: `60`

---

## 🛠️ Algorithm Steps (Logic)

1. **Input:** Ask the user to enter three numbers.
2. **Read:** Store the three values into variables (e.g., `Num1`, `Num2`, `Num3`).
3. **Processing:** - Create a variable named `Sum`.
   - Calculate the total using the formula: `Sum = Num1 + Num2 + Num3`.
4. **Output:** Print the value of the `Sum` variable to the screen.

---

## 📊 Flowchart Logic

1. **Start**
2. **Input:** `Read Num1, Num2, Num3`
3. **Process:** `Sum = Num1 + Num2 + Num3`
4. **Output:** `Print Sum`
5. **End**
